# Word格式自动调整功能使用指南

## 功能概述

RS-Paper-Writing Skill现在支持**一键Word格式调整**功能，可以根据你学校的毕业论文模板自动调整论文格式。

## 核心特性

✅ **一键格式调整** - 自动应用学校模板格式
✅ **保持内容完整** - 只改变样式，不改变内容
✅ **多模板支持** - 支持多个学校的预设模板
✅ **自定义规则** - 支持自定义格式规则
✅ **批量处理** - 支持批量处理多个文档
✅ **格式预览** - 支持格式预览和验证

## 快速开始

### 第1步：准备学校模板

你需要提供以下信息之一：

**选项A：提供Word模板文件**
- 上传你学校的毕业论文模板（.docx格式）
- 我会自动提取格式规范

**选项B：提供格式规范文档**
- 学校的毕业论文格式要求说明
- 包括标题、正文、页边距等详细规范

### 第2步：使用格式调整功能

```bash
# 基本用法
bash scripts/format_word.sh <input_file> <template_name> [output_file]

# 示例1：使用默认模板
bash scripts/format_word.sh thesis.docx default

# 示例2：使用学校模板
bash scripts/format_word.sh thesis.docx your_school_name

# 示例3：指定输出文件
bash scripts/format_word.sh thesis.docx your_school_name formatted_thesis.docx
```

### 第3步：在Skill中使用

直接告诉Skill你的需求：

```
我的论文已经写好了，现在需要按照我们学校的毕业论文格式调整。
我的学校是[学校名称]，论文文件是[文件路径]。
能否帮我一键调整格式？
```

Skill会自动：
1. 加载你学校的模板配置
2. 读取你的论文文档
3. 应用格式规范
4. 生成格式调整后的Word文档

## 支持的格式调整项

### 页面设置
- ✓ 页面大小（A4、Letter等）
- ✓ 页边距（上下左右）
- ✓ 页眉页脚

### 标题格式
- ✓ 一级标题（字体、字号、加粗、颜色、对齐）
- ✓ 二级标题
- ✓ 三级标题
- ✓ 标题间距

### 正文格式
- ✓ 字体和字号
- ✓ 行距
- ✓ 段落间距
- ✓ 首行缩进
- ✓ 对齐方式

### 特殊格式
- ✓ 图表标题
- ✓ 表格格式

- ✓ 页码和页眉页脚

## 模板配置说明

### 配置文件结构

```json
{
  "name": "template_name",
  "school": "School Name",
  "description": "Template description",

  // 页面设置
  "page_width": 8.5,
  "page_height": 11.0,
  "margin_top": 1.0,
  "margin_bottom": 1.0,
  "margin_left": 1.25,
  "margin_right": 1.25,

  // 正文样式
  "body_font": {
    "name": "宋体",
    "size": 12,
    "bold": false,
    "italic": false,
    "color": null
  },

  // 其他格式...
}
```

### 字体名称参考

- 中文字体：宋体、黑体、楷体、仿宋、微软雅黑
- 英文字体：Times New Roman、Arial、Calibri

### 颜色格式

- 十六进制：`"#000000"`（黑色）
- RGB元组：`"(0, 0, 0)"`

## 常见场景

### 场景1：为论文应用学校模板

```
我的论文已经完成，现在需要按照学校的格式要求调整。
学校是[学校名称]，论文文件是thesis.docx。
```

**Skill会：**
1. 加载学校模板配置
2. 自动调整所有格式
3. 生成thesis_formatted.docx

### 场景2：批量处理多个文档

```
我有5篇论文需要按照学校格式调整，都在documents/文件夹中。
```

**Skill会：**
1. 遍历所有Word文档
2. 逐个应用格式
3. 生成格式调整后的文档

### 场景3：自定义格式规则

```
我需要自定义一些格式规则：
- 标题用黑体18号
- 正文用宋体11号
- 行距1.5倍
```

**Skill会：**
1. 创建自定义模板配置
2. 应用你的规则
3. 保存为新模板供后续使用

## 常见问题

**Q: 格式调整会改变论文内容吗？**
A: 不会。只改变样式和格式，内容完全保持不变。

**Q: 支持哪些学校的模板？**
A: 目前支持自定义模板。你提供格式规范后，我会为你创建模板。

**Q: 可以同时调整多个文档吗？**
A: 可以。支持批量处理。

**Q: 调整后的文档可以继续编辑吗？**
A: 完全可以。输出的是标准Word文档，可以继续编辑。
